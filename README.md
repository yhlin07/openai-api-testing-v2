# OpenAI API Testing

This repository is used for testing OpenAI API functionalities.

## Contents

- `openai_setup.py`: Basic OpenAI API setup and usage example
- `requirements.txt`: Required dependencies

## Usage

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Run the example script:
   ```
   python openai_setup.py
   ```
